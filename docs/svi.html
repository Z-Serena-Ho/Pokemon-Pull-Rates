This is the first set of the Scarlet & Violet Era for the Pokemon Trading Card Game. It is commonly referred to as the Base Set for the Era and features the signature Pokemon of Generation IX, Koraidon and Miraidon. A couple other highlights from the set are Mirriam and the Ralts Evolution line.
